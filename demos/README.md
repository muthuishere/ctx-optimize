# Demos

VHS tape scripts for the site's terminal recordings
(https://muthuishere.github.io/ctx-optimize-site/). Render with
[vhs](https://github.com/charmbracelet/vhs): `vhs 01-add.tape` from this dir.
Tapes reference machine-local corpus paths in their hidden setup — adapt
`common.sh` and the `Hide` blocks before rendering elsewhere. Recordings are
unedited; if a demo can't be reproduced by running the same commands, that's
a bug.
