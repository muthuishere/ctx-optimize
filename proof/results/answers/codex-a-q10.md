Writeback throttling uses `wbt_wait()` in `block/blk-wbt.c`.

A bio is throttled only if `bio_to_wbt_flags()` marks it `WBT_TRACKED`: WBT enabled, and the bio is a write or discard, excluding `WRITE_ODIRECT` detected as `(REQ_SYNC | REQ_IDLE)` ([block/blk-wbt.c](/private/tmp/claude-501/-Users-muthuishere-muthu-gitworkspace-nexus-workspace-brain/dc4417dd-0040-4de7-82a4-10f10843dc78/scratchpad/proof/linux/block/blk-wbt.c:601), [block/blk-wbt.c](/private/tmp/claude-501/-Users-muthuishere-muthu-gitworkspace-nexus-workspace-brain/dc4417dd-0040-4de7-82a4-10f10843dc78/scratchpad/proof/linux/block/blk-wbt.c:619)). `wbt_wait()` returns immediately unless `WBT_TRACKED` is set; otherwise it calls `__wbt_wait()` ([block/blk-wbt.c](/private/tmp/claude-501/-Users-muthuishere-muthu-gitworkspace-nexus-workspace-brain/dc4417dd-0040-4de7-82a4-10f10843dc78/scratchpad/proof/linux/block/blk-wbt.c:646)).

The actual wait decision is in `rq_qos_wait()`: it tries `wbt_inflight_cb()`, which permits progress only if `rq_wait_inc_below(rqw, get_limit(...))` succeeds ([block/blk-wbt.c](/private/tmp/claude-501/-Users-muthuishere-muthu-gitworkspace-nexus-workspace-brain/dc4417dd-0040-4de7-82a4-10f10843dc78/scratchpad/proof/linux/block/blk-wbt.c:572)). If that cannot acquire an inflight slot below the limit, `rq_qos_wait()` queues the task and sleeps with `io_schedule()` until woken ([block/blk-rq-qos.c](/private/tmp/claude-501/-Users-muthuishere-muthu-gitworkspace-nexus-workspace-brain/dc4417dd-0040-4de7-82a4-10f10843dc78/scratchpad/proof/linux/block/blk-rq-qos.c:254)).

The rq_qos hook is `rq_qos_ops.throttle`: WBT installs `.throttle = wbt_wait` ([block/blk-wbt.c](/private/tmp/claude-501/-Users-muthuishere-muthu-gitworkspace-nexus-workspace-brain/dc4417dd-0040-4de7-82a4-10f10843dc78/scratchpad/proof/linux/block/blk-wbt.c:946)).
